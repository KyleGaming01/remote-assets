## 🔥 3. ASSET DATABASE SYSTEM

Create a local persistent database using ScriptableObject OR JSON.

Each asset must contain:

- key (string identifier)
- url (Firebase URL)
- type (audio/texture)
- local cache path
- last used timestamp

Example:

{
  "Reload_AK47": {
    "url": "...",
    "type": "audio"
  }
}
